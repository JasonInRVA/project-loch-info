What this document is

This is the main January 29, 2026 joint permit application volume submitted for Project Loch. It is the core applicant package rather than an agency decision.

Why it matters

This file organizes the permit request itself: project identity, site information, application forms, alternatives discussion, and the permit framework used by DEQ, VMRC, and USACE. It is the starting point for understanding what the applicant asked regulators to approve.

What the document says

The application describes Project Loch as a roughly 351-acre project area in Chesterfield County. It includes the permit forms, applicant and property information, and the impact-inventory structure used for surface waters and wetlands. It also points reviewers toward the supporting technical material in the companion environmental-report volume.

How to read it

Because this is an applicant submission, its descriptions of project purpose, design choices, avoidance, and mitigation are applicant assertions. The file is essential, but it should be read together with the DEQ additional-information requests, the VMRC non-jurisdiction letter, and the later July 2026 mitigation correspondence.

What it does not establish

This package does not prove that the application was complete, acceptable, or approved as submitted. Later DEQ letters show that regulators sought additional information and that the mitigation approach changed during review.
